# MouseTrap 项目简介

## 项目概述
MouseTrap 是一个基于 Java Swing 的迷宫游戏应用程序，旨在为用户提供交互式的迷宫生成与解决体验。该项目采用递归回溯算法生成随机迷宫，并使用广度优先搜索算法自动寻找最短路径解决方案。

## 核心目标
该代码库的核心目标是创建一个可玩的迷宫游戏，玩家可以控制老鼠在迷宫中移动，寻找奶酪作为目标。项目解决了迷宫生成、路径查找和图形界面交互等关键技术问题，提供了多种难度级别的迷宫选择。

## 技术栈
- **编程语言**: Java
- **图形界面**: Swing 框架
- **核心算法**: 
  - 递归回溯算法：用于生成随机的完美迷宫（无环路）
  - 广度优先搜索：用于寻找从起点到终点的最短路径
- **设计模式**: 面向对象设计

## 项目结构
项目采用模块化设计，包含以下主要组件：

### 核心组件
- [`MazeGenerator.java`](src/MazeGenerator.java:1) - 迷宫生成和路径查找核心逻辑
- [`MouseTrap.java`](src/MouseTrap.java:1) - 游戏主控制器，处理用户输入和游戏状态
- [`MazeSelectionMenu.java`](src/MazeSelectionMenu.java:1) - 迷宫选择界面，提供多种难度选项
- [`Frame.java`](src/Frame.java:1) - 主窗口框架组件
- [`Graphics.java`](src/Graphics.java:1) - 图形界面渲染组件
- [`Cell.java`](src/Cell.java:1) - 迷宫单元格数据结构

### 数据模型
- **Cell 类**: 表示迷宫单元格，包含上下左右四个方向的墙壁状态

## 功能特性
1. **迷宫生成**: 支持多种尺寸的迷宫生成（5x5、9x9、4x6、18x8）
2. **游戏交互**: 玩家通过方向键控制老鼠移动
3. **路径显示**: 可显示最短路径解决方案
4. **难度选择**: 提供不同难度级别的迷宫选择
5. **游戏状态管理**: 包含重新开始、新建迷宫等功能

## 部署支持
项目支持 Docker 容器化部署，包含 X11 图形界面支持，便于在不同环境中运行。

## 设计特点
- 各组件职责明确，便于维护和扩展
- 采用面向对象设计原则
- 图形界面响应式设计
- 算法实现高效且可读性强

可以通过以下方式运行项目：
1. 直接运行 `MouseTrap.java` 文件
2. 使用命令行编译运行
3. 通过 Docker 容器运行
