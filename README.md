# MouseTrap

### About
- Generates a maze using **Recursive Backtracking**.
- Finds the shortest solution path using **Breadth-first search (BFS)**.
- Maze is playable.

## How to Play
1. Choose how large of a maze you want to play
2. Control the mouse using the arrow keys
3. Reach the cheese to win
4. Use the "Show Solution" button if you get stumped
5. Have fun!

## How to Run

To start, either:
1. Run the file `MouseTrap.java` from your favorite IDE, or
2. From the command-line, type:
 ```sh
cd /<path_to>/MouseTrap/
javac src/*.java
java -cp src MouseTrap
 ```
